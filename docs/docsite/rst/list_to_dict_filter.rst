felixfontein.tools.list_to_dict -- Convert a list of values and keys to a dictionary
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

``[1, 2] | felixfontein.tools.list_to_dict(['a', 'b'])`` evaluates to ``{'a': 1, 'b': 2}``.
